# 在线表情包生成器

一个简单易用的在线表情包生成器，允许用户上传图片并在其上添加文字，生成个性化的表情包。

## 功能特点

- 支持本地图片上传（JPG、PNG、GIF）
- 实时图片预览
- 可在图片上方、中间和下方添加文字
- 自定义字体大小和颜色
- 一键下载生成的表情包为PNG格式
- 响应式设计，适配移动设备
- 简洁美观的用户界面

## 使用方法

1. 点击"选择图片"按钮上传本地图片
2. 在顶部、中间和底部输入框中输入想要添加的文字
3. 调整字体大小和颜色
4. 点击"生成表情包"预览效果
5. 点击"下载表情包"保存为PNG文件

## 技术栈

- HTML5
- CSS3
- JavaScript (原生，无框架)
